# agri-bt-robot-page

This folder contains a static site for the AGRI-BT Robot paper project.

How to use:
- Copy your manuscript PDF to `assets/manuscript.pdf`.
- Replace the title, authors, and abstract in `index.html`.
- Add figures into `assets/` and update image filenames in `index.html`.
- Publish with GitHub Pages by committing this folder to a repository and enabling Pages.
